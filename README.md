# Meme Generator

A simple web-based meme generator built with p5.js.

## Features

- Upload an image to use as the base
- Add top and bottom text
- Customize text color, size, and font family
- Drag text to position it anywhere on the image
- Draw shapes (circles, rectangles) or free draw on the image
- Save the resulting meme as a PNG image
- Share on social media (Twitter, Reddit, Facebook)

## How to Use

1. Open `index.html` in a web browser.
2. Upload an image using the file input.
3. Enter top and bottom text.
4. Adjust text properties as needed.
5. Drag text to reposition.
6. Use drawing tools if desired.
7. Click "Save Meme" to download the image.
8. Use share buttons to share on social media.

## Technologies Used

- p5.js for canvas manipulation
- HTML5 and CSS for UI
- JavaScript for interactivity

## Bonus Features Implemented

- Font family selection
- Social media sharing (opens share URLs in new tabs)
- Draggable text
- Drawing shapes and free drawing

## Troubleshooting

- Ensure your browser supports modern JavaScript features.
- For sharing, note that data URLs may not work on all social media platforms; consider uploading the saved image manually.
